# Badminton-Scoreboard
Badminton scoreboard build using Arduino UNO and MAX7219

Used libraries:<br>
  -LedControl.h<br>
  -LiquidCrystal_I2C.h<br>
  -wire.h<br>

Program designed for:<br>
  -Arduino UNO<br>
  -LCD i2c 1602 display<br>
  -5 pushbuttons<br>
  -6 - 7 segment common cathode displays<br>
  -MAX 7219 chip
