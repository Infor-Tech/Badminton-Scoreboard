# Badminton-Scoreboard
Badminton scoreboard build using Arduino UNO and MAX7219

<b>Used libraries:</b>
<ul>
  <li>LedControl.h</li>
  <li>LiquidCrystal_I2C.h</li>
  <li>wire.h</li>
</ul>

<b>Program designed for:</b>
<ul>
  <li>Arduino UNO</li>
  <li>LCD i2c 1602 display</li>
  <li>5 pushbuttons</li>
  <li>6 - 7 segment common cathode displays</li>
  <li>MAX 7219 chip</li>
</ul>
